# Ubuntu下安装软件

## .bundle安装包

.bundle是linux中的一种可执行文件。类似Windows的.exe

```
# 修改目标.bundle的执行权限
chmod +x xxx.bundle
# 运行安装包
sudo ./xxx.bundle
```
